- 使用 **记忆系统**（Memory）：在对话中让我保存重要信息到项目的 `.claude/memory/` 目录，这样下次打开项目时我可以读取这些记忆

**总结**
- 不主动保存 = 关闭 VS Code 或新开聊天 = 忘记
